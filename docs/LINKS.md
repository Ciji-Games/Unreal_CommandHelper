# Links

The Links tab provides quick access to Unreal Engine resources, grouped by category.

## Categories

| Category | Contents |
|----------|----------|
| **Official links** | Unreal Engine forums, documentation, Fab.com |
| **Community links** | Unreal Garden, asset naming conventions, Unreal Source Discord |
| **Ciji Games** | Gamedev backend, French UE courses, FAB listings, C++ Generator, LD Assistant 2 |

> [!TIP]
> Each link opens in your default browser. The links are configurable in the app's source (`src/config.ts`).
